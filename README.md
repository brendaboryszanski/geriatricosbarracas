# geriatricosbarracas.com.ar

Static website for Geriátricos Barracas (Le-Sa & Rincón del Sur), migrated from a PHP hosting to Vercel.

## Stack

- Static HTML + CSS + JS
- WOWSlider v8.7 (jQuery-based image slider)
- Bootstrap 2

## Deploy

Deployed on Vercel. No build step required — Vercel serves the static files directly.

## Contact

The contact section displays phone numbers and email. No form backend needed.
